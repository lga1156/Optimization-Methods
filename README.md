# Birds Sort Puzzle — решение головоломки

Этот проект представляет собой реализацию алгоритма для решения игровой головоломки "Birds sort Puzzle", в рамках курсовой работы по предмету «Методы оптимизации». Программа способна как находить оптимальную последовательность ходов для решения головоломки, так и проверять корректность уже существующего решения.

---

## Постановка задачи

Игровое поле состоит из нескольких вертикальных колонок (веток), частично заполненных птицами разных видов. Виды птиц обозначаются заглавными латинскими буквами `(A, B, C...)`. Несколько колонок изначально пусты и служат в качестве буфера для перемещений.

**Правила перемещения**:
- Перемещать можно только **крайнюю** птицу из любой непустой колонки.
- Птицу можно переместить в другую колонку, только если:
    - Целевая колонка **пуста**.
    - Крайняя птица в целевой колонке **того же вида**, и в колонке есть свободное место.

Задача считается **решенной**, когда каждая непустая колонка содержит только птиц одного вида и полностью заполнена. Все остальные колонки должны быть пустыми.

## Основные возможности

### 1. Validator (Валидатор входных данных)

Программа-валидатор проверяет корректность исходных данных задачи перед началом решения.

**Что проверяется**:
- ✅ **Единообразие длины веток**: Все заполненные колонки должны иметь одинаковую длину
- ✅ **Кратность количества птиц**: Количество птиц каждого вида должно быть кратно длине ветки
- ✅ **Структура файла**: Корректность формата входных данных (секция DATA, пустые ветки `==`, завершение `/`)

**Функциональность**:
- Автоматический поиск всех файлов в директории `data/`
- Подробный анализ каждого файла с выводом статистики:
  - Общее количество веток (заполненных и пустых)
  - Количество птиц каждого вида
  - Длина веток
  - Результат проверки кратности
  - Первая обнаруженная ошибка (если есть)
- Сохранение результатов в `logs/validation.log`

**Пример вывода**:
```
============================================================
Проверка файла: ../data/BIRDS_5.txt
→ Анализ структуры файла...
✅ Структура файла корректна (блоки найдены).
✅ Количество веток: 13 (норма ≤ 1000)
→ Проверка содержимого блока DATA...
✅ Все ветки одинаковой длины: N = 10
✅ Количество птиц каждого типа кратно N.
✅ Все проверки пройдены успешно.
Результат: OK
============================================================
```

**Запуск исполняемого файла**:
```bash
cd build
./validate
```
